





**工具的使用场景：**

* 每次修改完代码后，是否正在经历反复地打包？

* 在 Maven 、Git 以及其他运维脚本和工具的之间频繁切换？

* 采用 SCP 工具上传？使用XShell或SecureCRT登陆服务器？替换部署包？重启？

* 文件上传到服务器指定目录，在各种 FTP、SCP 工具之间频繁切换 ？
