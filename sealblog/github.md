## 1.Octotree

在开发一个项目的时候，会创建很多包使代码结构层次分明。

拿 Java 项目来说，一般会创建 src 目录用来存放项目源码，test 目录存放一些用来测试的脚本。在 src 目录下，还会有 resource 资源和一些业务逻辑代码（比如遵循 MVC 开发模式的Controller、Service、Dao ）
