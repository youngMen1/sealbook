# 1.Nacos是什么
Nacos 致力于帮助您发现、配置和管理微服务。Nacos 提供了一组简单易用的特性集，帮助您快速实现动态服务发现、服务配置、服务元数据及流量管理。 是Spring Cloud A 中的服务注册发现组件，类似于Consul、Eureka，同时它又提供了分布式配置中心的功能，这点和Consul的config类似，支持热加载。
Nacos 的关键特性包括:
* 服务发现和服务健康监测
* 动态配置服务，带管理界面，支持丰富的配置维度。
* 动态 DNS 服务
* 服务及其元数据管理
