# 1.windows环境启动

#### Nacos

```
windows系统点击startup.cmd就可以启动nacos
```

#### Sentinel

```
// 在cmd命令窗口
java -jar sentinel-dashboard-1.6.1.jar
```

## Linux环境启动

#### Nacos

linux或mac执行以下命令启动nacos

```
sh startup.sh -m standalone
```

#### Sentinel

linux或mac执行以下命令启动Sentinel

```
java -jar sentinel-dashboard-1.6.1.jar
```
