# 1.基本介绍 

## 1.1.filter的作用和生命周期 

由filter工作流程点，可以知道filter有着非常重要的作用，

在“Predict”类型的过滤器可以做参数校验、权限校验、流量监控、日志输出、协议转换等，

在“post”类型的过滤器中可以做响应内容、响应头的修改，日志的输出，流量监控等。首先需要弄清一点为什么需要网关这一层，这就不得不说下filter的作用了。
