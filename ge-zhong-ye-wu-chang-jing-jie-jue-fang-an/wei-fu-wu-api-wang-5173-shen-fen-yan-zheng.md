# 1.微服务API网关-身份验证

## 身份验证介绍

身份验证通常是进入系统的第一道大闸，要求用户出具登录此系统的身份证明。
其实在实际情况下，很多人开发的系统都没有身份验证功能或者就只有普通用户名和密码验证功能，这样的系统其实都是不完善的，也是不符合安全标准的，所以作为这么大众化的系统，我们有必要做详细的学习和了解。

## 身份验证分类
对于身份验证，我们分两种情况来看待，**界面登录验证**和**接口调用验证**，这两种方式所使用的身份验证方法很多时候是不一样的。
