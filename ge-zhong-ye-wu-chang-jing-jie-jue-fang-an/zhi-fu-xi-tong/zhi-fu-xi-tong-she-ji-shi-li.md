## 1. XXPay支付系统设计示例

### 支付渠道
| **渠道ID **| **对应渠道名称** |
| :--- | :--- |
| WX\_JSAPI | 微信公众号支付 |
| WX\_NATIVE | 微信原生扫码支付 |
| WX\_APP | 微信APP支付 |
| WX\_MWEB | 微信H5支付 |
| ALIPAY\_MOBILE | 支付宝移动支付 |
| ALIPAY\_PC | 支付宝PC支付 |
| ALIPAY\_WAP | 支付宝WAP支付 |
| IAP | 苹果应用内支付 |
